Pure.css

A set of small, responsive CSS modules that you can use in every web project.

Get Started

Key Features

Lightweight

With a tiny 4.5kb filesize, this set of CSS modules makes a realistic, simple starting point for development with CSS.

Responsive Layouts

Responsive by deisign and made with mobile in mind. Done using a grid system similar to Bootstrap, Pure.css also has a non-responsive option.

Modular

Take what you need by using any comination of modules that you need or simply take the style that you need straight from the source.

Plays Nice

Unlike other frameworks, Pure.css' design is unopinionated, minimal and flat, made with the philosophy it's much easier to add new CSS rules than to overwrite existing rules.

Frontend Freedom

Yahoo launched the Pure.css framework in mid-2013. Unlike Bootstrap, which is based on RWD principles, Pure.css works on SMACSS â€“ Scalable and Modular Architecture for CSS. Moreover, unlike its competitors, Pure.css is designed to be a starting point for a new project, not a complete solution.

Pure.css is a good candidate if you want a barebones frontend framework to build on top of. It is easy to customize and can be extended with select Bootstrap components to increase your productivity. You can also use it by itself for an easy to use, easy to load Twitter Bootstrap alternative.

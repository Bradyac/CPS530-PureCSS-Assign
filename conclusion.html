Conclusion

It is easy to say that the power of Pure.css is clear when working on this project. The easy installation process and intuitive starting properties make it an obvious choice if you are designing a website from the ground up. In our case, it was vital to the creation of this website and made the entire process much simpler than starting from scratch. With this website as evidence it is clear that Pure.css performed just as expected. Since Pure is regular css, it was easy to learn seeing as we already learned CSS previously. There are no extra complications, in fact it is easy to add, remove and change any existing styles further solidifying its idea to be a great starting point for any website.

No regrets - 5/5

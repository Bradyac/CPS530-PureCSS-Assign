CDN Installation Guide

Using a CDN for Pure is the easiest way to get up and running quickly. Similar to importing other css files it can simply be done using a link tag.

<link rel="stylesheet" href="https://unpkg.com/purecss@1.0.0/build/pure-min.css" integrity="sha384-nn4HPE8lTHyVtfCBi5yW9d20FjT8BJwUXyWZT9InLYax14RDjBj46LmSztkmNP9w" crossorigin="anonymous">

You can also import individual modules using the following links.

Name URL
Base https://unpkg.com/purecss@1.0.0/build/base-min.css
Buttons https://unpkg.com/purecss@1.0.0/build/buttons-min.css
Forms (Responsive) https://unpkg.com/purecss@1.0.0/build/forms-min.css
Forms (Non-Responsive) https://unpkg.com/purecss@1.0.0/build/forms-nr-min.css
Grids (Base) https://unpkg.com/purecss@1.0.0/build/grids-min.css
Grids (Responsive) https://unpkg.com/purecss@1.0.0/build/grids-responsive-min.css
Menus (Responsive) https://unpkg.com/purecss@1.0.0/build/menus-min.css
Tables https://unpkg.com/purecss@1.0.0/build/tables-min.css

NPM Installation Guide

Note that when using any frameworks through a CDN, the files that make-up the framework are immutable. This means in-order to take specific parts of Pure.css you would have to download the individual files and edit them based on what you need. This can be done easily with NPM install.

$ npm install purecss
